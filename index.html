About Me

My favourite two programming language are VB.NET and Java. It is very uncommon to see someone appreciate VB.NET much these days, but I have my own reasons. I grew up using it to create small computer applications as tools. In 2015, I started using ASP.NET and I found out that I really enjoyed web development. I took me almost no time to learn MSSQL and start integrating my web applications with it in order to expand my capabilities. On the other hand, I learnt Java during high school. I started appreciating this language when I finally took a Software Design course. I used Java to design explore several design patterns that extended my way of thinking from a coding and problem solving perspective. Aside from all of that, I love playing ping pong!
